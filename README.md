# secure-multi-tier-vpc-AWS
Designed and deployed a 3 tier AWS VPC architecture across 2 Availability Zones using public and private subnets. Implemented Application Load Balancer and Auto Scaling Groups scaling 2 to 4 instances. Deployed Amazon RDS MySQL in private subnets, used NAT Gateways, IAM roles, least privilege security groups, with CloudWatch monitoring SNS alerts.
